# Minimalistic Content Slider

## Use four block only:

1. Structural(HTML)
2. Styling(CSS)
3. Inputs(Events)
4. Motion Logic(JS)

### Structure(What exists)
```yml
Text Slider Wrapper (div)
- Text Slider (div)
  - Text Slide (div) [multiple]
    - Text (h1) -> (br)
Slider Control (div)
  - Prev (div)
    - button (button)
  - Next (div)
    - button (button)
Blocks (div)
  - block-[index] (div) [multiple]
Overlay (div)
Image Slider (div)
  - Image Slide (div) [multiple] {style background-image:url()}
```
